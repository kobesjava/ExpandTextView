# JAppendTextView
有两段文字组成  
第一段文字可以设置最大显示行数 超过了三个点显示  
第二段文字全部显示 demo中是（等待审核）  
如果需要支持span 可以参考JExpandTextView自己修改  
运行TextExpandListActivity看效果
